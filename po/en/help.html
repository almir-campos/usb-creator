USB Creator Help

Description

The USB Creator is a tool to help users to create a bootable flash drive with multiple Linux operating systems.

Important

Make sure you backed up all important data from the flash drive before you start.

Commands

You can also use the USB Creator from the terminal. Here are some examples:

# usb-creator -f -i /path/to/my.iso -m

This will format the flash drive, copy the ISO file to the flash drive and check the hash of the copied ISO file with the original ISO file.

# usb-creator -b -g -r

This will try to repair an unbootable flash drive that already contains ISO files.

# usb-creator -h

This will show all available options and their usage.

Hidden features

When you remove the ISO file name from the ISO field, USB Creator will calculate the space needed for all ISO files in that directory.

You can change the order in which the ISO files are displayed in the menu by adding a number in front of the ISO file name, e.g.: "01-MyDistro-amd64.iso".

Symbolic links are followed but the symbolic links must have the ".iso" extension to make it work.

Your ISO is not booting?

On our GitHub page you can see a list of tested distributions but not all can be booted directly from the ISO file. Check the distribution's forum if it can be booted directly from the ISO file.

The menu is configured in a file which you can find on your flash drive: /boot/grub/grub.cfg

Please help improve this application by adapting grub.cfg until the ISO boots and post your changes on GitHub or our forum.

Don't forget to attach the log file when you post an issue on our forum: /var/log/usb-creator.log
