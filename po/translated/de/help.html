USB Creator Hilfe

Beschreibung

Der USB Creator ist ein Werkzeug zur Herstellung eines startfähigen Flash-Laufwerks mit mehreren Linux-Betriebssystemen.

Wichtig

Alle wichtigen Daten auf dem Flash-Laufwerk müssen gesichert werden, bevor Sie beginnen.

Befehle

USB Creator kann auch vom Terminal aus bedient werden. Hier sind einige Beispiele:

# usb-creator -f -i /Pfad/zu/meinem.iso -m

Dies formatiert das Flash-Laufwerk, kopiert die ISO Datei auf das Laufwerk und vergleicht die hash-Prüfsummen.

# usb-creator -b -g -r

Versuch der Reparatur eines nicht startfähigen Flash-Laufwerks, auf dem sich bereit ein ISO befindet.

# usb-creator -h

Dieser Befehl zeigt verfügbaren Optionen und Ihre Benutzung.

Versteckte Eigenschaften

Wenn Sie den ISO-Dateinamen aus dem ISO Feld entfernen, wird USB Creator den notwendigen Platz für alle ISO-Dateien in diesem Verzeichnis berechnen.

Die Reihenfolge der im Menü angezeigen ISOs lässt sich durch das Hinzufügen einer Nummer vor dem ISO-Dateinamen ändern, wie z.B. "01-MeineDistro-amd64.iso".

Symbolische Links funktionieren unter der Voraussetzung, dass sie eine ".iso" Erweiterung haben.

Startet Ihr ISO nicht?

Auf unserer GitHub-Seite steht eine Liste der getesteten Linux-Distributionen, aber nicht alle können direkt von der ISO-Datei gestartet werden. Dies können Sie im Forum der Distribution überprüfen.

Das Menü kann über eine Datei konfiguriert werden, die Sie auf Ihrem Flash-Laufwerk unter /boot/grub/grub.cfg finden.

Bitte helfen Sie, indem Sie die grub.cfg solange anpassen, bis das ISO bootet, und uns Ihre Verbesserungen auf GitHub oder in unserem Forum mitteilen.

Sollten Sie ein Frage in unserem Forum stellen, dann vergessen Sie bitte nicht die Log-Datei unter /var/log/usb-creator.log anzuhängen.
