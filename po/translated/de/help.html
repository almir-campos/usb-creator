USB Creator Hilfe

Beschreibung

Der USB Creator ist ein Werkzeug, mit dessen Hilfe ein bootfähiges Flash-Laufwerk mit mehreren Linux-Betriebssysteme erstellt werden kann.

Wichtig

Bevor Sie beginnen, sollten Sie sicherstellen daß alle wichtigen Daten auf dem Flash-Laufwerk gesichert wurden.

Befehle

Sie können USB Creator auch vom Terminal aus bedienen. Hier sind einige Beispiele:

# usb-creator -f -i /path/to/my.iso -m

Damit wird das Flash-Laufwerk formatiert, die ISO Datei auf das Laufwerk kopiert und ein md5-Prüfsummenvergleich initialisiert.

# usb-creator -b -g -r

Dieser Befehl führt einen Reparaturversuch an einem nicht bootbaren Flash-Laufwerk durch, auf dem sich bereit eine ISO Datei bedindet.

# usb-creator -h

Dieser Befehl wird alle verfügbaren Optionen und Ihre Benutzung anzeigen.

Versteckte Eigenschaften

Wenn Sie den ISO Dateinamen aus dem ISO Feld entfernen, wird USB Creator den notwendigen Platz für alle ISO Dateien in diesem Verzeichnis berechnen.

Die Reihenfolge der im Menü angezeigen ISO Dateien lässt sich durch das Hinzufügen einer Nummer vor dem eigentlichen ISO Dateinamen erreichen, wie z.B. "01-MeineDistro-amd64.iso"

Symbolische Links funktionieren unter der Voraussetzung, dass die eine ".iso" Erweiterung haben.

Bootet Ihre ISO Datei nicht?

Auf unserer GitHub Seite können Sie eine Liste der getesteten Linux-Distributionen einsehen, aber nicht alle können von können direkt von der ISO Datei gebootet werden. Schauen Sie im Forum nach, ob die Distribution direkt von der ISO Datei gebootet werden kann.

Die Konfiguration des Menüs lässt sich über eine Datei bewerkstelligen, die Sie auf Ihrem Flash Laufwerk unter /boot/grub/grub.cfg finden.

Bitte helfen Sie diese Anwendung zu verbessern, indem Sie die grub.cfg Datei solange anpassen, bis die ISO Datei bootet. Bitte teilen Sie uns Ihre Verbesserungen auf GitHub oder in unserem Forum mit.

Sollten Sie ein Frage in unserem Forum stellen, dann vergessen Sie bitte nicht die Log-Datei unter /var/log/usb-creator.log anzuhängen.
