Aide d'USB Creator

Description

USB Creator est un outil pour aider les utilisateurs à créer une clé USB amorçable contenant plusieurs systèmes d'exploitation Linux.

Important

Assurez-vous d'avoir sauvegardé toutes vos données importantes provenant de la clé USB avant de commencer.

Commandes

Vous pouvez aussi utiliser USB Creator à partir du terminal. Voici quelques exemples:

# usb-creator -f -i /chemin/vers/mon.iso -m

Cela va formater la clé USB, copier l'ISO sur cette clé et vérifier la somme hash de l'ISO copiée avec celle de l'ISO originale.

# usb-creator -b -g -r

Cela va tenter de réparer une clé USB non amorçable qui contient déjà les fichiers ISO.

# usb-creator -h

Cela vous présente toutes les options disponibles et leur utilisation.

Fonctions cachées

Lorsque vous enlevez le nom de l'ISO de son champ, USB Creator calculera l'espace requis pour tous les ISO de ce répertoire.

Vous pouvez modifier l'ordre dans lequel s'affichent les ISO dans le menu en ajoutant un nombre devant le nom de l'ISO, par ex.: "01-MaDistro-amd64.iso".

Les liens symboliques sont suivis mais ils doivent avoir une extension ".iso" pour fonctionner.

Votre ISO ne s'amorçe pas?

Sur notre page GitHub vous pouvez trouver une liste de distributions testées mais toutes ne peuvent pas être amorçées directement par leur ISO. Vérifiez le forum de cette distribution si tel est le cas.

Le menu est configuré par un fichier que vous pouvez trouver sur la clé USB: /boot/grub/grub.cfg

Merci de nous aider à améliorer ce programme en modifiant grub.cfg jusqu'à ce que l'ISO s'amorce et postez vos modifications sur GitHub ou notre forum.

N'oubliez pas d'ajouter le fichier de log lorsque vous postez une question sur notre forum: /var/log/usb-creator.log
