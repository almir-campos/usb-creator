USB Creator Help

Descrizione

USB Creator è uno strumento per facilitare la creazione di dispositivi avviabili con più sistemi operativi Linux.

Importante

Assicurarsi di aver salvato tutti i dati importanti dalla periferica perima di incominciare.

Comandi

È anche possibile utilizzare USB Creator dal terminale. Di seguito alcuni esempi:

# usb-creator -f -i /path/to/my.iso -m

Questa azione formatterà il dispositivo, copierà l'immagine ISO e controllerà l'md5sum dell'immagine ISO copiata con l'immagine originale.

# usb-creator -b -g -r

Questa azione proverà a riparare un dispositivo non avviabile con immagini ISO preesistenti.

# usb-creator -h

Mostra ogni opzione disponibile e le relative modalità d'uso.

Opzioni nascoste

Rimuovendo il nome del file ISO dal campo ISO, USB Creator calcolerà lo spazio necessario alle immagini ISO in tale cartella.

È possibile modificare l'odine in cui i file ISO sono mostrati nel menu aggiungendo un numero prima del nome del file ISO, ad es.: "01-LaMiaDistro-amd64.iso".

I collegamenti simbolici verrano seguiti ma devono avere l'estensione ".iso" per funzionare correttamente.

L'immagine ISO non si avvia?

Sulla nostra pagina di GitHub è pubblicata una lista di distribuzioni testate ma non tutte possono essere avviate direttamente dall'immagine ISO. Controllare il forum della distribuzione per accertarsi che sia direttamente avviabile da immagine ISO.

Menu configurato grazie ad un file disponibile sul dispositivo: /boot/grub/grub.cfg

Si prega di migliorare questa applicazione adattando il contenuto di grub.cgf fino a che l'immagine ISO sia correttamente avviabile e quindi di condividere le modifiche necessarie su GitHub o sul nostro forum.

Ricordarsi di allegare il file di log quando si intende chiedere supporto sul nostro forum: /var/log/usb-creator.log
