USB Creator ヘルプ

説明

USB Creator は、複数の Linux オペレーティングシステムが起動できるフラッシュドライブを作成するのに役立つツールです。

重要

開始前に、フラッシュドライブにある重要なデータをすべてバックアップして下さい。

コマンド

ターミナルから USB Creator を使用することができます。ここでいくつかの例を示します。

# usb-creator -f -i /path/to/my.iso -m

フラッシュドライブをフォーマットし、ISO ファイルをコピーした後、オリジナルの ISO ファイルとコピーした ISO ファイルとの hash チェックサムを確認します。

# usb-creator -b -g -r

すでに ISO ファイルが含まれている起動できないフラッシュドライブの修復を試みます。

# usb-creator -h

すべての利用可能なオプションとその使用方法を表示します。

隠し機能

ISO フィールドから ISO ファイル名を削除すると、USB Creator はそのディレクトリ内にあるすべての ISO ファイルに必要なスペースを計算します。

ISO ファイルは、ISO ファイル名の前に数を追加することによって、メニューに表示される順序を変更することができます。例えば: "01-MyDistro-amd64.iso"

シンボリックリンクは動作します。ただし動作させるためには ".iso" 拡張子にしている必要があります。

ISO が起動しない？

GitHub ページでは、テスト済みのディストリビューション一覧を参照できますが、その ISO ファイルすべてが直接起動できるわけではありません。ISO ファイルから直接起動できるかディストリビューションのフォーラムを確認して下さい。

メニューは、フラッシュドライブにある次のファイルで設定を見ることができます: /boot/grub/grub.cfg

ISO を起動する grub.cfg の適合によって、このアプリケーションを改善したい場合は GitHub または フォーラム で変更内容を投稿してください

フォーラムに問題を投稿する場合、ログファイルの添付を忘れないで下さい: /var/log/usb-creator.log
