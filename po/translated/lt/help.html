USB kūrėjo pagalba

Aprašas

USB kūrėjas yra įrankis, skirtas padėti naudotojams sukurti paleidžiamą USB atmintuką su keliomis Linux operacinėmis sistemomis.

Svarbu

Prieš pradėdami, įsitikinkite, kad padarėte visų savo svarbių duomenų, iš USB atmintuko, atsarginę kopiją.

Komandos

Jūs taip pat galite naudotis USB kūrėju terminale. Štai yra keli pavyzdžiai:

# usb-creator -f -i /kelias/į/mano.iso -m

Tai formatuos USB atmintuką, nukopijuos į atmintuką ISO failą ir palygins nukopijuoto ir pradinio ISO failo maišą.

# usb-creator -b -g -r

Tai pabandys pataisyti nepasileidžiantį atmintuką, kuriame jau yra ISO failas.

# usb-creator -h

Tai parodys visas prieinamas parinktis ir jų naudojimą.

Paslėptos ypatybės

Jei pašalinsite ISO failo pavadinimą iš ISO lauko, USB kūrėjas apskaičiuos visiems, tame kataloge esantiems failams, reikiamą laisvą vietą.

Jūs galite keisti tvarką, kuria meniu yra rodomi ISO failai, pridėdami ISO failo pavadinimo pradžioje skaičių, pvz., "01-ManoDistribucija-amd64.iso".

Simbolinių nuorodų yra paisoma, tačiau tam, kad veiktų, simbolinės nuorodos privalo turėti ".iso" plėtinį.

Jūsų ISO nepasileidžia?

Mūsų GitHub puslapyje galite matyti išbandytų distribucijų sąrašą, tačiau jos ne visos gali būti paleidžiamos tiesiogiai iš ISO failo. Išsamiau, žiūrėkite distribucijos forume, kad sužinotumėte ar distribucija gali būti paleista tiesiogiai iš ISO failo.

Meniu yra sukonfigūruotas faile, kurį rasite savo atmintuke: /boot/grub/grub.cfg

Prašome padėti mums tobulinti šią programą, pritaikant grub.cfg iki to, kai pasileidžia ISO ir paskelbiant pakeitimus puslapyje GitHub arba mūsų forume.

Paskelbdami mūsų forume apie problemą, nepamirškite pridėti žurnalo failą: /var/log/usb-creator.log
