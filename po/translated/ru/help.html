Помощник USB Creator

Описание

USB Creator это инструмент, который поможет Вам создать загрузочный флэш-диск с операционной системой Linux.

Важный

Прежде чем начать убедитесь что Вы сделали резервные копии всех важных данныёх с флеш-накопителя.

Команды

Вы также можете использовать USB-Creator из терминала. Вот несколько примеров:

# usb-creator -f -i /path/to/my.iso -m

Это позволит форматировать флешку, скопировать ISO файл на флешку и проверить hash сумму с оригинального ISO-файла.

# usb-creator -b -g -r

Исправить незагружающийся флеш-диск содержащий ISO образ.

# usb-creator -h

Это покажет все доступные варианты и их использования.

Скрытые возможности

Когда Вы удалите имя файла ISO из поля ISO, USB Creator вычислит пространство, необходимое для всех файлов ISO в этом каталоге.

Вы можете изменять порядок вывода файлов ISO в меню на экране, добавляя число перед именем файла ISO. Пример: "01-MyDistro-amd64.iso".

Символические ссылки поддерживаются, но они должны иметь расширение ".iso" чтобы работать.

Ваш ISO не загрузочный?

На странице GitHub Вы можете посмотреть список протестированных дистрибутивов, но не все они могут быть загружены напрямую с файла ISO. Проверьте на форуме дистрибутива, может ли он быть загружен непосредственно с ISO.

Меню сконфигурировано в файле, который Вы можете найти на Вашем флеш-накопителе: /boot/grub/grub.cfg

Помогите улучшить это приложение, адаптируя grub.cfg для загрузочных ISO и публикуя изменения на GitHub или нашем форуме.

Не забудьте приложить файл журнала, когда размещаете вопрос на нашем форуме: /var/log/usb-creator.log
